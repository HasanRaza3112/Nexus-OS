# Architecture Decision Records (ADR)

Document important architectural and technical decisions made for this project.

## ADR-0001 — Scope Narrowing to Unity Dev Launch Service
Date: 2026-06-27
Decision: Initial business focus narrowed from a 15-category service catalog to 3 Unity development services (bug fixing, feature development, performance optimization), based on having zero paid clients and no proof of work outside Unity projects.
Status: Accepted

## ADR-0002 — Documentation Structure
Date: 2026-06-27
Decision: The existing numbered folder scaffold (00_Project_Management through 07_QA) is the single source of truth for all project documentation. No parallel /docs structure will be created.
Status: Accepted
